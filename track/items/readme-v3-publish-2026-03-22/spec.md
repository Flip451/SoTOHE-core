## Goal

README.md を v3 ビジョンに合わせて更新し、公開可能な状態にする。

## Scope

### In Scope

- README.md の v3 ビジョン反映（変更は実施済み）

- コードの変更
- 他のドキュメントの変更

- [ ] README.md が main にマージされている
