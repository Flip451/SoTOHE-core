# Knowledge Directory

Consolidated project knowledge base. All documentation, research, conventions,
and architecture decisions live here.

## Reading Order

1. `conventions/README.md` — Project-specific engineering rules and implementation policies
2. `adr/README.md` — Architecture Decision Records
3. `WORKFLOW.md` — Development workflow details
4. `external/POLICY.md` — External long-form guide operating policy
5. `external/guides.json` — External guide registry
6. `strategy/TODO-PLAN.md` — Strategic roadmap

## Directory Structure

| Directory | Purpose |
|-----------|---------|
| `conventions/` | Project-specific implementation rules |
| `adr/` | Architecture Decision Records |

| `designs/` | Design documents (auto-mode, etc.) |
| `schemas/` | Schema documentation |
| `external/` | External guide policy and registry |
| `strategy/` | Strategic planning and roadmap |
